◈ .tblx — Base-8 Cipher Format

How it works

The .tblx format encodes all data using a Base-8 (octal positional) letter substitution cipher. Each letter A–Z maps to its position in octal: A=0, B=1, … H=7, I=10, J=11, … Q=20, … Z=31. Numbers and punctuation pass through as-is. Spaces are preserved.

Y30Z31Lowercase same as uppercase · Numbers/symbols unchanged

Line 1: TBLX_V1                    ← magic header
Line 2: <base8-encoded JSON blob>  ← all data encrypted
