◈ .tblx — Base-8 Cipher Format

How it works

The .tblx format encodes all data with a Base-8 (octal positional) letter-substitution cipher. Each letter A–Z maps to its alphabetical position written in octal: A=0, B=1… H=7, then I=10, J=11… Q=20… Z=31. Numbers and symbols pass through unchanged.

Cipher Reference

A0B1C2D3E4F5G6H7
I10J11K12L13M14N15O16P17
Q20R21S22T23U24V25W26X27
Y30Z31Lowercase same · Numbers/symbols unchanged

Example

Plaintext:  NO      → Ciphertext: 1516
Plaintext:  Hello   → Ciphertext: 74131316
Plaintext:  TBLX    → Ciphertext: 23 1 13 27
