◈ Format Reference

.tblx — Base-8 Cipher

All text is encoded with a Base-8 (octal) letter-substitution cipher. Each letter A–Z maps to its alphabetical index in octal: A=0, B=1 … H=7, I=10, J=11 … Z=31. Numbers and symbols pass through unchanged.

Cipher Map

A0B1C2D3E4F5G6H7
I10J11K12L13M14N15O16P17
Q20R21S22T23U24V25W26X27
Y30Z31Lowercase maps same · Numbers/symbols unchanged

.tbly — Styled Bundle

A .tbly file is a ZIP archive (renamed) containing two files: data.tblx (the base-8 encoded table data) and theme.css (auto-generated stylesheet matching your table's visual configuration — colors, fonts, layout). Use TBL.loadBundle() to unpack both assets in one call.

.tbly file structure

data.tbly/
├── data.tblx   ← base-8 cipher encoded table
└── theme.css   ← generated stylesheet (colors, fonts, layout)
